Sprotty Animation & Transitions Showcase

This example demonstrates Sprotty's comprehensive animation system with various animation types, state transitions, and performance optimizations.

Animation Recipe

Interactive Controls

  • Click Animation Nodes: Each node triggers its own animation (bounce-node bounces, pulse-node pulses, shake-node shakes, glow-node glows, complex-node spins)
  • Click State Nodes: Cycle through different states (idle, processing, complete, error) with smooth color transitions
  • Shift+Click Complex Node: Trigger a compound animation combining multiple effects
  • Click Edges: Start animated flow effects along the connections
  • Press Escape: Stop all running animations

Animation System Features

  • Triggered Animations: User interactions trigger smooth, responsive animations with customizable easing functions and durations
  • State Transitions: Visual elements smoothly transition between different states with interpolated colors and properties
  • Compound Animations: Multiple animations can be combined and choreographed for complex, coordinated visual effects
  • Performance Optimized: Built on browser's animation frame for smooth 60fps performance with monitoring and optimization tools
  • CSS Integration: Seamlessly combines Sprotty animations with CSS animations for enhanced visual effects
  • Accessibility Aware: Respects user preferences for reduced motion and provides alternative interaction methods

Animation Types Demonstrated

  • Bounce: Elements jump with elastic easing using a decay function
  • Pulse: Scaling effect that oscillates in and out
  • Shake: Rapid horizontal movement with decreasing amplitude
  • Spin: Full 360-degree rotation animation
  • Glow: Pulsing glow effect using SVG filters
  • Edge Flow: Animated data flow along connection edges
